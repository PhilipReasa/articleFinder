isNodeWhitespace(HTML)

Takes in a HTML node and returns true if it is "whitespace". Whitespace is defined as:

  • a non text node AND
  • the node.data is not regex whitespace

shouldIgnoreElementBySelfOrAncestorTagName(element) → {boolean}

Determines if the element passed in via parameter element should be ignored based. There are two cases that the tag may be ignored:

  1. The tag was present in the const CandidateTagNamesToIgnore
  2. One of the tags parents was in CandidateTagNamesToIgnoreDescendantsOf
